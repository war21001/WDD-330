Week 4 Notes

Forms

Form Element

  • method called elements that return an HTML collection of all elements contained in the form
  • acess using 'name' attribute

Properties and Methods

  • form.submit() - submits the form automatically
  • ex. - button with type submit
  • form.reset()
  • <button> type ='reset'>Reset</button>
  • form.action property can be used to set the action attribute of a form to a different URL
  • form.action='/an/other.url'

Events

  • focus - event occurs when an element is focused on (curosor inside of field)
  • blur - focus away from an element
  • change - moves focus away after changing it

Submitting a Form

  • Submit event - send content of the form to the server to be processed.
  • Use JavaScript to personalize submit event
  • preventDefault() - prevents form from submitting

Retrieving and Changing Values from a form

  • Text input element objects have a value property that can be used to retrieve the text inside the field
  • Can set in JavaScript - input.value = "Search here";
  • focus and blue events to clear out text
  • Placeholder attribute - not actual value of field

Form Controls

  • <input> - text, passwords, check boxes, radio buttons, file uploade
  • <select> - menus for drop down lists of options
  • <textarea> - elements for longer text entry
  • <button> - elements for submitting and resetting forms

Input Fields

  • type = 'text', default is text
  • password - type = 'password', secret information, characters are concealed as they are entered
  • checkbox - type='checkbox', checked or not, more than one can be selected
  • radio button - type = 'radio', only one option can be selected
  • Hidden - type = 'hidden', not displayed in browser. NOT secret. Value can be changed in JavaScript
  • File Input - type = 'file', used to upload files and button
  • Other - number, tel and color
  • Select Drop Down List - <select> one or more options can be selected
  • Text Area - <textarea> , long pieces of text
  • Buttons - <button>, default is submit
