Regular Expressions

  • A pattern that can be used to search strings for matches to the pattern => find and replace
  • Create=> /[a-z] [A-Z] + int$/;
  • Sequence uses the dash
  • \^ negates the sequence
  • Properties
    • global (g)- Return all matches
    • Ignore case (i) -pattern is case insensitive, default is case sensitive
    • multi line (m) - make pattern on multiple line
  • Special Characters
    • . Matches any character, except line breaks
    • \w matches any word, an is equivalent to [A-Z a-z 0-9 ]
    • \W Matches nonword characters [\^ A-Z a-z 0-9 ]
    • \d Digits [0-9]
    • \D non digits [\^ 0-9]
    • \s white space [\t \r \n \f]
    • \S [\^ \t \r \n \f]
  • Modifiers - placed after token to deal with multiple occurrences
    • ? Optional
    • * Matches one or more occurrences preceding
    • + matches one or more currents is proceeding
    • {n} matches at matches n occurrences of pattern
    • {n,} matches one or more n occurrence is a pattern
    • {,m} matches at most M occurrences preceding token
    • {n,m} matches at least n but no more than M occurrences
    • ^ marks the position immediately before the first character in a string
    • $ marks the position immediately after the last character
    • when needed use the\ to escape out

Events

  • Event listeners tells the program when an event has occurred such as a mouse click, keypress, screen touch
  • multiple functions can be attached independently two different events
  • Whenever an event handler is triggered by an invent the callback function is called. Function is automatically passed on an event object as a parameter that contains information about the event
  • Event propagation - the order that events fire on each element
    • Bubbling - Fires element clicked on first then bubbles up the document tree, firing an event on each parent element until it reaches the root node
    • Capturing- starts by firing an event on the root element, then propagates downward, firing each child element until it reaches the target element that was clicked on see
