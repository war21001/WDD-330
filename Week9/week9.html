Window Object

The Browser Object

  • Collection of properties and methods that contain infomation about the browser and computer screen
  • No official standard
  • number of properites and methods taht are supported by all major browsers, making a sort of de facto standard
  • available through window object
  • Node.js does not have a window object - global
  • Going Global
    • global variables - const, let and var keywords
    • global variables are part of a global object - >any global variles are part fo the window object
  • Dialogs
    • alert(), confirm(), prompt()
    • stops the program until user responds

Browser Information

  • the window object has a number of properties and methods that porvide information about the user's browser
  • window.navigator.userAgent - returns info about the browser and operating system being used
  • can be modified
  • location, location,location
    • window.location - object that contains info about the URL of the current page
    • properties - href, portocol, host, hostname, port, portname, search, has, origin
    • methods - reload(). replace(), assign(), toString()

Browser History

  • windows.history property - used to access previously visited pages in the current browser session
  • windows.history.length - shows how many pages have been visited before arriving at the current page
  • window.history.go() - used to got to a specific page, ..where() - is the current page, ..forward(), ..back() - navigate forward & backward

Controlling Windows

  • open() - opens new windo ->URL, window title, list attributes
  • close() - used to close a window
  • moveTo() - move the window in the screen
  • resizeTo() - resize the window

Screen Information

  • window.screen - info about the screen the browser is displayed on
  • .screen.height & .width - dimensions
  • .screen.availHeight & .availWidth
  • colorDepth - color bit depth of the user's monitor

Document Object

  • contained in each window object
  • document.write() - write string, replaces what was previously on page
  • don't use

Cookies

  • small files saved locally on a user's computer
  • restriction is they can only be read by the web page from the same domain that set them
  • limited to 4KB data
  • Used to personalize a browser's experience
    • store user preferences
    • tracker user choices
    • autheniticate
    • track users
    • retain state
  • text file that contains a list of name-vailue pairs seperated by semi colons
